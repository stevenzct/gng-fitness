# GNG Fitness Member Hub Prototype

A mobile-first HTML/CSS prototype for a gym website and member management system.

## Included sections

- Landing page
- Membership plans
- Online inquiry form
- Trainer profiles
- Trainer booking preview
- QR check-in preview
- Member/admin dashboard preview
- Promo/referral section
- FAQ

## How to run

Open `index.html` directly in your browser.

## Suggested next step

Convert this prototype into Next.js, Nuxt, or React once the gym approves the design direction.
